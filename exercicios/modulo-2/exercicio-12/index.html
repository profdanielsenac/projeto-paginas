Cores em CSS

Para entender esse exercício, olhe bem atentamente o código fonte do arquivo.
Foi colocado um gradiente ao fundo usando a propriedade background-image: linear-gradient().

Cores por Nome

Essa caixa e seus textos foram coloridos usando cores por nome, o nome darkblue e o nome white, respectivamente.

Cores por Código Hexadecimal

Essa caixa e seus textos foram coloridos usando códigos hexadecimais, o código #00008b e o código #ffffff, respectivamente.

Códigos RGB (red, green, blue)

Essa caixa e seus textos foram coloridos usando códigos RGB, o código rgb(0, 0, 139) e o código rgb(255, 255, 255), respectivamente.

Códigos HSL (hue, saturation, light)

Essa caixa e seus textos foram coloridos usando códigos HSL, o código hsl(240, 100%, 27%) e o código hsl(0, 0%, 100%), respectivamente.

Transparência

Nesse caixa, o fundo azul teve sua transparência configurada para 50% usando o formato RGBA (red, green, blue, alpha), com o código rgba(0, 0, 139, 0.5).

As cores resultantes são exatamente as mesmas, mas o formato de representação muda conforme o padrão escolhido.
